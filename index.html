Redmine TPA
Tags for Performance Appraisal

Redmine TPA (Tags for Performance Appraisal) is a redmine plugin that helps Performance Appraisals. It makes possible to link issues to pre-established criteria or organizational objectives.

Redmine TPA (Tags for Performance Appraisal) é um plugin redmine que auxilia avaliações de desempenho. Ele torna possível a ligação de tarefas à critérios pré-estabelecidos ou objetivos organizacionais.

Link criteria to issues
Una critérios à tarefas

Procedure:
Open an issue, click on "Marcar para Avaliação" (pt-BR), select an appraisal and click on the tag icon located to the left of a criteria that you wish to link to the issue.
Procedimento:
Abra uma tarefa, clique em "Marcar para Avaliação", selecione uma avaliação e clique no icone de etiqueta localizado a esquerda do criterio que voce deseja unir à tarefa.
