Redmine TPA
Tags for Personal Appraisal

Redmine TPA (Tag for Performance Appraisal) é um plugin redmine que auxilia avaliações de desempenho. Ele torna possível a criação de marcações que unem tarefas a critérios pré-estabelecidos ou objetivos organizacionais.

Redmine TPA (Tag for Performance Appraisal) is a redmine plugin that helps Performance Appraisals. It makes possible to create tags that link issues to pre-established criteria or organizational objectives.
